# e-plantShopping

## Project Description
An e-commerce plant shopping application built with React and Redux. This application allows users to browse various plant categories, add plants to their shopping cart, and manage their cart items.

## Features
- **Product Listing**: Browse plants by categories (Air Purifying, Aromatic Fragrant, Insect Repellent, Medicinal, Low Maintenance)
- **Shopping Cart**: Add, remove, and update quantities of plants in the cart
- **Real-time Updates**: Cart total updates dynamically as items are added/removed
- **Responsive Design**: User-friendly interface for plant shopping

## Technologies Used
- React.js
- Redux Toolkit (for state management)
- React Hooks (useState, useEffect, useSelector, useDispatch)
- CSS for styling

## Project Structure
- `ProductList.jsx` - Main product listing component
- `CartItem.jsx` - Shopping cart component
- `CartSlice.jsx` - Redux slice for cart state management
- `store.js` - Redux store configuration

## How to Run
1. Clone the repository
2. Install dependencies: `npm install`
3. Start the development server: `npm start`
4. Open http://localhost:3000 in your browser

## Functionality
- Add plants to cart from the product listing
- View and manage cart items
- Update quantities (increment/decrement)
- Remove items from cart
- Calculate total cart amount
